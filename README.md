# Middle space

Qu'est-ce que 'Middle space' ?

Middle space est une application web qui permet de creer un espace où les client peuvent trouver rapidement les entreprises qui pourront répondre à leur besoins 

# Langage de programmation et Framework
* Rubis version 3.0.1
* Rails version 6.0.6
# Comment faire fonctionner le site Web
* git clone https://github.com/alass77/middleApp.git
* cd cd middleApp
* bundle install
* yarn install --check-files
* rails db:create && rails db:migrate
* rails s
